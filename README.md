## Github

- **Daily Git Commands**
    git status
    git add .
    git commit -m "message" 
    git push

- **Create a Branch** 
    git checkout -b feature/branch

- **Push a Branch** 
    git push -u origin feature/branch

- **Pull Requests** 
    gh pr create --title "My Feature" --body "Description"

- **List PRs** 
    gh pr list

- **Check Out a PR** 
    gh pr checkout 123

- **Create Issues** 
    gh issue create --title "Bug: Something broke" --body "Details..."

- **List Issues** 
    gh issue list
